## Overview

LuigiC is a native C AI engine for the Super Mario World C-reimplementation.

Instead of trial-and-error learning, it uses real-time physics simulation and A* search to calculate thousands of potential futures every frame. 

This allows for autonomous, frame-perfect navigation with high physics accuracy while maintaining a stable 60 FPS depending on hardware.
