## Overview

LuigiC is a native C AI engine for the Super Mario World C-reimplementation. 

It utilizes real-time state simulation and A* search to identify optimal navigation paths through complex physics environments. 

The engine is designed for stable 60 FPS execution, with search depth and path precision scaling dynamically based on available hardware overhead.
